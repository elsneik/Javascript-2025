Javascript es un lenguaje de progamacion dinamico de alto nivel para crear interfaces interactivas y dinamicas.

es uno de los 3 pilares fundamentales del desarollo web.

  • Html - define la estructura del contenido. (elementos que aparecen en la pagina
  • Css - Define la apariencia (colores. tamaños y estilos)
  • Javascript - define el comportamiento ( lo que hace la pagina o sus elementos)

caracteristicas de Javascript

  1. Lenguaje interpretado no necesitamos copilarlo. el navegador lo entiende directamente.
  2. Basado en objetos - maneja objetos como document, windows, arrays, date, class
  3. Dinamico - podemos modificar el contenido en tiempo real
  4. Versatil y de facil aprendizaje - aunque nacio para la web, se usa tambien en servidores, aplicaciones moviles, videojuegos y hoy en dia en Inteligencia artificial
